# 🐍 Python Beginner Projects

A collection of small, self-contained Python programs built while learning the language. Each script focuses on a different core concept — loops, conditionals, functions, file handling, and simple algorithms — through fun, practical mini-projects.

## 📋 Table of Contents

## 📖 About

This repository is a personal learning log — a set of beginner-friendly Python projects covering games, utilities, and small tools. It's meant to demonstrate fundamental programming concepts like loops, conditionals, functions, string manipulation, and basic file I/O.

## 🗂️ Projects

| Project | Description |
|---|---|
| [`atm.py`](atm.py) | A simple ATM simulator supporting deposits, withdrawals, and balance checks |
| [`cows_and_bulls_game.py`](cows_and_bulls_game.py) | Classic number-guessing logic game with "cows" and "bulls" hints |
| [`currency-converter.py`](currency-converter.py) | Converts an amount between different currencies |
| [`dice_rolling_game.py`](dice_rolling_game.py) | Simulates rolling a dice |
| [`number_guessing_game.py`](number_guessing_game.py) | Guess the randomly generated number within a limited number of tries |
| [`password_generator.py`](password_generator.py) | Generates random, secure passwords |
| [`password_strength_checker.py`](password_strength_checker.py) | Checks how strong a given password is |
| [`pig_dice_game.py`](pig_dice_game.py) | The classic "Pig" dice game with turn-based scoring |
| [`qr_code_generator.py`](qr_code_generator.py) | Generates a QR code from user input |
| [`quiz_game.py`](quiz_game.py) | A simple multiple-choice quiz game |
| [`rock_paper_scissor.py`](rock_paper_scissor.py) | Play Rock, Paper, Scissors against the computer |
| [`simple_text_editor.py`](simple_text_editor.py) | A basic command-line text editor |
| [`slot_machine.py`](slot_machine.py) | A slot machine simulation game |
| [`tic_tac_toe.py`](tic_tac_toe.py) | Classic two-player Tic Tac Toe |
| [`todo_list.py`](todo_list.py) | A command-line to-do list manager |
| [`word_guessing_game.py`](word_guessing_game.py) | Word-guessing game using a word list from [`words.txt`](words.txt) |
